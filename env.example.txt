# Google Drive Storage Configuration
# Follow these steps to set up Google Drive storage:
#
# 1. Go to https://console.cloud.google.com/
# 2. Create a new project (or select existing)
# 3. Enable the Google Drive API:
#    - Go to APIs & Services > Library
#    - Search for "Google Drive API"
#    - Click Enable
#
# 4. Create a Service Account:
#    - Go to IAM & Admin > Service Accounts
#    - Click "Create Service Account"
#    - Give it a name like "nullgen-storage"
#    - Click Create and Continue
#    - Skip the optional steps, click Done
#
# 5. Create a JSON key:
#    - Click on the service account you created
#    - Go to Keys tab
#    - Add Key > Create new key > JSON
#    - Download the JSON file
#
# 6. Share your Google Drive folder:
#    - Create a folder in Google Drive for NullGen uploads
#    - Right-click > Share
#    - Add the service account email (from the JSON file, "client_email" field)
#    - Give it "Editor" permission
#
# 7. Get the folder ID:
#    - Open the folder in Google Drive
#    - The URL will be like: https://drive.google.com/drive/folders/FOLDER_ID_HERE
#    - Copy the FOLDER_ID_HERE part

# Service Account Email (from JSON file)
GOOGLE_CLIENT_EMAIL=your-service-account@your-project.iam.gserviceaccount.com

# Private Key (from JSON file - keep the quotes and \n characters)

# Google Drive Folder ID (from folder URL)
GOOGLE_DRIVE_FOLDER_ID=your-folder-id-here
